# ISGT2025-Stochastic-model-for-planning-standalone-energy-community-under-extreme-events
Optimization of a Standalone Energy Community with Vehicle-to-Community and Load Shifting under Extreme Events


ISGT Latam 2025 - Optimization of a Standalone Energy Community with Vehicle-to-Community and Load Shifting under Extreme Events

Prepared by: Dayara Pereira Basso

Supervised by: John Fredy Franco Baquero

FAPESP Process no.: 2024/20610-2.

Model implemented in AMPL and solved with CPLEX.

".mod" files represent the model and constraints;

".dat" files represent the parameters and data;

".run" file represents the main script to execute the model with the data;

".svg" files are figures of the results obtained with the test data.
